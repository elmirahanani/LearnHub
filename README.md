## Deskripsi Karya
platform edukasi online namanya LearnHub Isinya ada empat halaman utama: Beranda, About Us, katalog kursus "Smart Learning", sama halaman Contact, lengkap sekalian sama versi mobile-nya biar responsif.
Buat konsepnya, di sini saya pakai gaya yang modern dan playful dengan main di gradasi warna biru-ungu biar kelihatan fresh. Tata letaknya juga sudah dibuat rapi pakai bantuan ilustrasi 3D dan ikon-ikon simpel supaya navigasi pas belajarnya nanti lebih gampang dan nggak ngebosenin buat user.
